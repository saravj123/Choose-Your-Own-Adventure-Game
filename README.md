# Choose-Your-Own-Adventure-Game

An interactive choose your own adventure game where you play as a thief trying to steal back jewels from a kingdom.

## Features
- Multiple story paths and endings
- Stealth vs combat decisions
- Interactive buttons and dynamic scenes
- Visual storytelling with images

## How to Play
Click through the story and make choices to reach different endings.

## Tech Used
- HTML
- CSS
- JavaScript
